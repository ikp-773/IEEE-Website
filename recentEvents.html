Get the Git

IEEE Computer Society Student Branch in association with the Department of Computer Science and Engineering (NETSEC Research Group) of Mar Baselios College of Engineering and Technology conducted an online webinar on “Introduction of Git and GitHub” on 1st of September,2020. The webinar was driven by Mr. Athul Cyriac Ajay, who is a GitHub Campus Expert and about 80 students attended the webinar. The webinar primarily focused on git commands and git branches. He talked about student development achievements in GitHub and helped the participants to start a git account. This webinar was very effective for students and the event ended off with a doubt clearance section. The event iwas absolutely a great success
